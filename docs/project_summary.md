# Project Summary

## Title
Workforce Retention Risk Analytics: A Predictive and Decision-Support Framework for HR Strategy

## Purpose
This project extends a standard HR employee attrition prediction task into a portfolio-ready decision-support case. The goal is to help HR stakeholders identify workforce risk patterns, understand possible drivers of turnover, and prioritize retention interventions.

## Business Question
How can HR identify workforce risk patterns early enough to improve employee retention, reduce overwork-related attrition, and prioritize managerial interventions?

## Core Value Add
The portfolio version adds:

1. HR KPI design
2. Risk segmentation
3. Ethical analytics framing
4. Business recommendation layer
5. Future Tableau dashboard plan
6. Clear distinction from the original course capstone

## Suggested Final Deliverables

- Portfolio website page
- Enhanced Python notebook
- Tableau dashboard
- One-page executive summary
- Responsible analytics note
- GADA vs. portfolio-enhanced comparison table
