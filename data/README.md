# Data Folder

This folder is reserved for dataset-related files used in the Workforce Retention Risk Analytics project.

The original dataset comes from the Google Advanced Data Analytics capstone project and is used for educational and portfolio demonstration purposes.

## Dataset Description

The dataset contains employee-level HR records, including satisfaction level, evaluation score, workload, tenure, promotion history, department, salary category, and whether the employee left the company.

## Public Release Note

Depending on data-sharing permissions and portfolio presentation needs, the public repository may include either:

- the original dataset,
- a cleaned sample dataset,
- a simulated or anonymized version, or
- documentation only.

This folder is maintained to separate data files from code, documentation, and dashboard materials.
