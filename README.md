# Workforce Retention Risk Analytics

This repository contains a portfolio-enhanced version of the Google Advanced Data Analytics capstone-style HR attrition project. Instead of presenting the work only as a course modeling assignment, this version reframes it as a workforce retention risk analytics and decision-support case.

## Project Goal

Build a predictive and decision-support framework that helps HR stakeholders identify employee attrition risk, workload pressure, promotion gaps, and practical retention priorities.

## Portfolio Positioning

The project demonstrates:

- Business problem framing
- PACE workflow application
- Exploratory data analysis and data quality review
- Predictive modeling for binary classification
- Risk segmentation for stakeholder decision-making
- Ethical and responsible analytics considerations
- Future Tableau dashboard planning

## Public Release Boundary

This project is for educational and professional portfolio demonstration. It should not be presented as a production HR system. Predictive outputs should be interpreted as decision-support signals, not as definitive claims about individual employees.
